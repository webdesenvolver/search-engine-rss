🔍 Search Engine RSS

Sistema de Busca com Google CSE + RSS Feeds

📋 Quick Start

1️⃣ Permissões de Arquivo

Execute os comandos abaixo para garantir que os diretórios tenham permissão de escrita:

chmod -R 777 data/
chmod -R 777 logs/
2️⃣ Configurar Google CSE

Edite config.php e adicione suas chaves:

define('GOOGLE_CSE_API_KEY', 'sua_chave_aqui');
define('GOOGLE_CSE_ID', 'seu_cse_id_aqui');
📖 Como obter as chaves
3️⃣ Testar Busca

Acesse a página principal e tente fazer uma busca:

🔗 Ir para Busca
4️⃣ Gerenciar RSS Feeds

Acesse o painel administrativo para adicionar feeds:

🔗 Painel Admin
5️⃣ Configurar Cron (Opcional)

Para atualizar feeds automaticamente, adicione ao crontab:

0 * * * * php -f /caminho/para/cron/fetch-feeds.php
⚠️ Execute crontab -e e adicione a linha acima

🐛 Problemas Comuns

❌ "Google CSE não configurado"

Solução: Edite config.php e adicione suas chaves de API

❌ "Permissão negada"

Solução: Execute chmod -R 777 data/ logs/

❌ "Feeds não atualizam"

Solução: Verifique se o cron está configurado corretamente e se as URLs de feed são válidas

Para mais ajuda, veja Troubleshooting na documentação

✅ Checklist de Instalação

  • ☑️ Permissões de arquivo configuradas
  • ☑️ Google CSE configurado
  • ☑️ Busca testada
  • ☑️ Feeds RSS adicionados
  • ☑️ Cron job configurado
